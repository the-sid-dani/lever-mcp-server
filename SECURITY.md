# Security Policy

## Supported Versions

Only version 2.x on the `main` branch is currently supported for security fixes.

## Reporting a Vulnerability

Email Sid Dani at sid.dani@samba.tv with the subject line:
`Lever MCP Security Report`

Do not open public GitHub issues for security reports.

## Response Expectations

Reports will receive acknowledgement within 48 hours and an initial assessment
within 7 days.

## Scope

In scope:

- Source code in this repository
- Deployed Cloud Run service at lever-mcp-201626763325.us-central1.run.app

Out of scope:

- Lever ATS itself; report those issues to Lever
- Auth0; report those issues to Auth0
- GCP infrastructure; report those issues to security@samba.tv
